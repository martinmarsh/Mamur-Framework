Class mamurDataObject

Description
Description | Methods (details)

mamurDataObject encapsulates data in an array structure.

The concept is that instead of using session arrays similar data items can be grouped into meaningful data objects which can be persisted to session data or to an ORM or database. mamurDataObject can be a single data item or a list of items accessed using $mydataObjectClass->$myVariableName object magic methods. Also dataobjects can support mutiple rows to form a table There are methods to switch rows and to index through row items. Note unlike using an ORM everything is stored in memory so it is designed for use of small datasets or subsets of data from a query.
